3DCurator

Un visor 3D de TC de esculturas

3DCurator es un software para visualizar datos volumétricos de esculturas de madera policromada a las que se les ha sometido a una Tomografía Computacional (TC). Incluye una serie de herramientas para llevar a cabo las tareas de documentación básicas que necesitan los restauradores.

El software es parte del TFG y TFM de las titulaciones de Grado en Ingeniería Informática y Máster en Ingeniería Informática de la UGR realizado por Francisco Javier Bolívar Lupiáñez y supervisado por Francisco Javier Melero Rus.

Se utilizan las librerías de alto nivel: Qt, VTK, ITK, OpenCV y Boost sobre C++ como lenguaje de programación. Se usa CMake para compilarlo en distintas plataformas.

Plataforma

Este software está disponible para equipos con Microsof Windows como sistema operativo

Librerías modernas

Se han usado las versiones más recientes de las librerías utilizadas a la hora de generar la última versión del software

Sencillo de utilizar

Nustro objetivo principal es desarrollar un software sencillo de usar para que cualquier usuario pueda ver los resultados deseados sin necesidad de ningún curso de formación

Libre y gratuito

El código fuente está alojado en un repositorio de GitHub y se puede descargar gratuitamente

Reconstrucción volumétrica

Realiza una reconstrucción volumétrica en 3D de la escultura usando disintos presets o creando el tuyo propio para visualizar los distintos materiales

Cortes

Realiza cortes con un plano arbitrario para poder ver el interior de la escultura y así descubrir elementos que no verías de ninguna otra forma sin dañar la figura

Funciones de transferencia

Proveemos distintos presets de funciones de transferencia para renderizar los materiales de esculturas de madera policromada (madera, estuco y metal), pero también puedes crear tu propia función de transferencia usando gráficas sencillas y exportarlas a formato XML para poder compartirlas con otros usuarios

Polygonal mesh

Extract the polygon mesh of any material in STL format for later printing in 3D

Filtering

Se proprocionan distintos filtros de reducción de ruido. Puedes usar los filtros media y gaussiano si quieres suavizar el volumen y el filtro mediana si te quieres deshacer del ruido de tipo sal y pimienta

Segmentación

¿Quieres dividir el volumen en las distintas piezas de madera que lo forman? Ahora puedes usar nuestro método semi automático para hacerlo

Documentación

Este software está hecho para restauradores y lo usarán para exampinar esculturas. Proporcionamos herramientas de documentación como medidores de distancias y ángulos o anotaciones. Pudes exportar todos estos elementos en formato XML para que tú o tus compañeros puedan importarlos y seguir trabajando con ellos
